User Guide & FAQ

How to Manage Inventory

To view, add, or remove items from the warehouse, navigate to the Management page. Here you will find a complete list of all products currently in stock.

Searching for Items

If the inventory list is too long, use the Search page. Simply type the name or ID of the product you are looking for.

Frequently Asked Questions (FAQ)

Q: Where is my data saved?

A: For this version, data is stored in the browser's memory. If you refresh the page, the list will reset to the default state.

Q: Can I edit an existing item?

A: Yes! Simply go to the Management page, where you can update the details of any item currently in the system.
