## Project Overview
In this part the inventory management system has been ported to Angular v17. The application is built as a Single Page Application (SPA) focusing on component-based architecture and centralized data management using Services.

## Application 
* **Home:** Welcome page with a system overview.
* **Management:** Page for creating, editing, and deleting inventory items.
* **Search:** Search functionality to find specific items in the database.
* **Security:** An analysis of application security and data integrity.
* **Help:** User guide.
